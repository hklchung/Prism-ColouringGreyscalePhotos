## About the Project
Convolutional Neural Networks (CNN) are commonly used for computer vision. In this project, I aim to build a CNN that can "predict" the colour layers of an image given the greyscale layer of the image.

## Dependencies
* Keras
* Scikit-Image
* PIL
* Tensorflow
* Matplotlib
